##  Projetos

### ₿ Bitcoin × S&P 500
Análise multiativos em tempo real comparando o índice S&P 500 (^GSPC) e o par BTC-USD, com coleta automatizada via Yahoo Finance.

- Nuvem de Ichimoku — suporte, resistência e direção de tendência
- EMA 20 — identificação de tendências de curto prazo
- Gráficos interativos com zoom, hover e exportação HTML via Plotly

**Stack:** Python · Pandas · Plotly · yfinance

 [github.com/renatosgk/BTC-X-SP500](https://github.com/renatosgk/BTC-X-SP500)

---

###  LUME — Sistema de Bem-Estar Emocional
Backend para monitoramento de bem-estar emocional e físico de trabalhadores — métricas fisiológicas via IoT, check-in emocional, relatórios inteligentes e autenticação segura via JWT. API documentada com Swagger.

- Spring Security + JWT (Auth0) para autenticação e autorização
- Integração com ThingSpeak (IoT) para leitura de métricas fisiológicas
- Documentação automática via Springdoc OpenAPI / Swagger
- Oracle Database com Spring Data JPA / Hibernate

**Stack:** Java · Spring Boot · Oracle · JWT · Swagger · Lombok

 [github.com/renatosgk/lume](https://github.com/renatosgk/lume)

---

###  OsFacil API — Ordens de Serviço Automotivas
Sistema de gerenciamento de ordens de serviço para o setor automotivo com arquitetura HATEOAS e banco Oracle.

**Stack:** Java · Spring Boot · Oracle · HATEOAS
